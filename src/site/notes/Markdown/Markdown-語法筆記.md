## 文字效果

- ~~槓掉、刪除線、`<s>`~~：`~~`
- <u>底線</u>：只能用`<u></u>`
- **粗體**：`**`
- _斜體_：`_` 或 `*`

## Code Related

- 可執行python：在code block上寫`run-python`
- 新增code檔案：`alt + v`
